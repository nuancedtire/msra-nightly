# About MSRA.uk

## Project Overview
MSRA.uk is a hobby project created by a resident doctor, for resident doctors. This website aims to provide organized study notes and resources for those preparing for the Multi-Specialty Recruitment Assessment (MSRA).

## Disclaimer
- This website is **not** officially affiliated with or endorsed by the MSRA, HEE, or any official examination body.
- All content is created based on personal study notes and publicly available resources.
- While we strive for accuracy, we cannot guarantee that all information is complete, current, or error-free.
- Always refer to official MSRA guidelines and resources for the most up-to-date and accurate information.
- This website should be used as a supplementary study resource only.
